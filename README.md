# Mesh Topology Simulation

Simulasi jaringan menggunakan topologi Mesh pada Cisco Packet Tracer.

## Deskripsi
Project ini menggunakan:
- 5 Switch
- 5 PC

Setiap switch saling terhubung satu sama lain sehingga membentuk koneksi penuh (full mesh).

## Konfigurasi
- Static IP Address
- Same Network ID
- Full Mesh Connection

## Implementasi
Topologi Mesh digunakan pada jaringan yang membutuhkan redundansi dan kestabilan koneksi tinggi.

## Kelebihan
- Memiliki banyak jalur cadangan
- Stabil dan minim gangguan
- Cocok untuk jaringan kritis

## Kekurangan
- Membutuhkan banyak kabel
- Konfigurasi lebih kompleks
- Biaya implementasi lebih tinggi
